VS Code Shortcuts

  • Open App in VS Code: code hotelapp
  • Navigate to Other Files: Alt + 1, Alt + 2, etc.
  • Navigate to a File: Ctrl + P and type the file name
  • Close the File: Ctrl + W
  • Close All Tabs: Ctrl + Shift + W
  • Minimize/Maximize Sidebar: Ctrl + B
  • Split Editor into Two Windows: Ctrl + \
  • Move to Next Editor Group: Ctrl + 1, Ctrl + 2, etc.
  • Delete Specific Line: Ctrl + X
  • Duplicate Lines: Alt + Shift + Down Arrow
  • Write Same Text in Multiple Places: Hold Alt and type
  • Move Lines Up/Down: Alt + Up Arrow / Alt + Down Arrow
  • Search for Code in File: Ctrl + O and type to search
  • Go to Specific Line: Ctrl + G and type :11
  • Remove Empty Lines: Use regex ^(\\s)*$\\n and Ctrl + Shift + Enter
